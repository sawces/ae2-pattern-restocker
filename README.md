# AE2 Pattern Restocker

A lightweight quality-of-life addon for [Applied Energistics 2](https://github.com/AppliedEnergistics/Applied-Energistics-2) that automatically refills the blank pattern slot in the Pattern Encoding Terminal using blank patterns already stored in your ME network.

No special cards, upgrades, or additional items required. As long as your ME system contains spare blank patterns, the terminal will automatically restock itself whenever the slot is empty and you click Encode.

## Why this mod?

Some larger AE2 addon mods include similar functionality, but also introduce many additional features, mechanics, and balance changes. This mod exists to provide only the auto-restock functionality in a minimal, standalone package that can be added to modpacks without significantly altering their intended experience.

## Compatibility

| Minecraft | NeoForge | AE2      |
|-----------|----------|----------|
| 26.1.2    | 26.1.x   | 26.1.x   |

## Installation

Drop the JAR into your `mods` folder alongside Applied Energistics 2. No configuration needed.
